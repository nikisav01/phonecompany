# Telephone Bill Calculator

Implementation of a telephone billing system that calculates call charges based on time-based pricing and promotional offers.

## Requirements

- Java 21
- Gradle 8.x

## Building the Project
```bash
./gradlew build
```

## Running Tests
```bash
./gradlew test
```

View test report:
```bash
./gradlew testReport
open build/reports/tests/test/index.html
```
